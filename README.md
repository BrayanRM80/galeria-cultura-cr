# Galería Fotográfica — Cultura Típica de Costa Rica

Aplicación web multimedia desarrollada con **Vue 3** como parte del proyecto personal del curso IF7102 Multimedios, I Ciclo 2026.

## Framework utilizado

Vue 3 (Composition API con Vite)

## Tema

Galería fotográfica interactiva sobre la cultura típica costarricense: comidas típicas, trajes típicos, artesanías y fiestas y tradiciones.

## Funcionalidades

- Filtros visuales CSS (escala de grises, sepia, contraste, saturación)
- Lightbox al hacer clic (sin librerías externas)
- Filtrado por categoría
- Metadata cargada desde JSON con fetch()
- Audio ambiental con control de volumen
- Diseño responsivo

## Cómo ejecutar

```bash
npm install
npm run dev
```

Abrí `http://localhost:5173` en tu navegador.
